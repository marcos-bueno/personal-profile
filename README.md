# Personal Profile

## Installation:
  - Install [Yarn](https://classic.yarnpkg.com/en/docs/install) (stable version);

Open your favorite Terminal and run these commands:

```sh
$ git clone git@github.com:marcos-bueno/personal-profile.git
$ cd personal-profile
$ yarn
$ yarn dev
```
